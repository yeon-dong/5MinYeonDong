## 기획

**연동(連動/聯動)**
「명사」
『기계』 기계나 장치 따위에서, 한 부분을 움직이면 연결되어 있는 다른 부분도 잇따라 함께 움직이는 일.
<br/>
위의 뜻과 같이 5분 동안 다른 사람들과 함께 주제에 대해 의견을 나눌 수 있는 공간을 만들고자 했습니다.
